# Patch_DBD
## 简介
黎明杀机【解锁帧数】和【解决当前杀鸡画质高糊】的问题（也就是关闭默认抗锯齿的效果）
## 注意
请勿二改 大量传播

**请关闭游戏后运行此脚本，运行一次后就不用再打开了**

**凡是提示拒绝访问的都是没有以管理员模式开启！！！**

原理是对游戏配置文件 *Engine.ini* 文件进行修改，也就是贴吧所流传的解锁帧数方法然后简化成脚本操作
